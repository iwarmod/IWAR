Un AddOn no oficial para el juego de PC "Arma" de Bohemia Interactive.
 
===================================== DESCRIPIÓN ===================================== 
Fuerzas irregulares equipadas con armamento y vehiculos del mod RHS Escalation. Por ahora, sólo existe una facción fictisia.

Compatible con ALiVE. Class name: usml_aif

===================================== REQUIRIMIENTOS ===================================== 
Arma 3 - version [1.52 o mayor] - Bohemia Interactive  
RHS: Armed Forces Of The Russian Federation - 0.3.9.1 - Red Hammer Studios ( http://http://rhsmods.org/ )
RHS: United States Armed Forces - 0.3.9.1 - Red Hammer Studios ( http://http://rhsmods.org/ )
